# ionic2-playground
ionic2 experimental and playground

# Getting started

## Install

### Automated
 - Open powershell within dir **as admin**
 - run `.\setup.ps1`
 - run `gulp watch`

### Manual
 - run `npm install`
 - create symbolic link in `wwwroot` to point to `node_modules`
 - run `gulp build`
 - run `gulp watch`


## Utils

### Build

```
gulp build
```

### Run/watch

```
gulp watch
```
